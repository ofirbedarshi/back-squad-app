# צ'קליסט החטאה

## תיאור
- טופס לתיעוד בדיקות אחרי החטאה.
- כל צ'קליסט נשמר כרשומה נפרדת עם זמן עדכון אחרון.
- רשימת הצ'קליסטים ממוינת לפי זמן העדכון האחרון — המעודכן ביותר מופיע ראשון.
- הנתונים נשמרים מקומית בלבד ולא נשלחים החוצה.

## חוקי חובה ושמירה
- אין כרגע שדות חובה בטופס.
- אפשר לשמור גם טופס חלקי או ריק.
- שדות בחירה מסוג כן/לא מקבלים ברירת מחדל: לא.
- שדות בחירה אחרים מקבלים את ברירת המחדל שמצוינת לידם.

## סוגי שדות
- **טקסט** — ערך חופשי.
- **מספר** — ערך מספרי.
- **בחירה** — בחירה מתוך אפשרויות קבועות.

- **זיהוי מקום הפגיעה** — בחירה: כן, לא; ברירת מחדל: לא.
  - אם **כן** — נפתחת בחירת סוג מיקום: קרוב, רחוק, תיאור מילולי, נ.צ.
  - **קרוב** / **רחוק** — אין שדות נוספים; הבחירה עצמה נשמרת.
  - **תיאור מילולי** — טקסט חופשי, אופציונלי.
  - **נ.צ.** — הזנת נ.צ. (מזרח, צפון, פלח), אופציונלי.
  - **האם התפוצץ** — בחירה: כן, לא; ברירת מחדל: לא.
  - אם **לא** — שדות המיקום לא מוצגים.
- **סוג המטרה** — טקסט, אופציונלי.
- **מטרה רפלקטיבית או משטחים רפלקטיביים בקרבתה** — בחירה: כן, לא; ברירת מחדל: לא.
- **מפנה המטרה** — טקסט, אופציונלי.
- **ריבוי כתמים** — בחירה: כן, לא; ברירת מחדל: לא.
- **הקפצת כתם** — בחירה: כן, לא; ברירת מחדל: לא.
- **הסטה** — בחירה: כן, לא; ברירת מחדל: לא.
- **מיקום הצלב ביחס לנקודת הפגיעה הרצויה** — טקסט, אופציונלי.
- **מיקום פגיעת הטיל ביחס לצלב** — טקסט, אופציונלי.
- **גלישה של הכתם** — בחירה: כן, לא; ברירת מחדל: לא.
- **בליעה של הכתם** — בחירה: כן, לא; ברירת מחדל: לא.

## שמירה מקומית
- הצ'קליסטים נשמרים במכשיר ונשארים לאחר סגירה ופתיחה מחדש של האפליקציה.
- ממסך רשימת הצ'קליסטים: לחיצה ארוכה על כרטיס פותחת תפריט; משם ניתן למחוק. המחיקה דורשת אישור, מסירה את הרשומה מהמכשיר ולא ניתן לשחזר מתוך האפליקציה.
