# מסך הערות

## תיאור
מסך לתזכורות בשטח: בכל הערה אפשר לשלב **טקסט** ו־**הקלטה קולית** יחד. אפשר גם רק טקסט וגם רק הקלטה. הטקסט נשמר במכשיר, וההקלטה נשמרת במאגר נפרד בדפדפן — הכל עובד בלי אינטרנט.

## הוספת הערה חדשה
- יש שני כפתורים "+ הוסף הערה" (בכותרת ובגוף המסך) — שניהם פותחים את אותו חלון
- בחלון: שדה טקסט (לא חובה אם יש הקלטה) ושורת הקלטה קולית קומפקטית עם כפתורי סימון
- אפשר לשמור רק אחרי שיש **לפחות** טקסט (לא ריק) **או** הקלטה שהסתיימה בהצלחה
- "שמור" בראש החלון שומר את ההערה, "ביטול" סוגר בלי לשמור

## רשימת ההערות
- ההערות מסודרות מהחדשה לישנה
- ליד התאריך מופיע משך ההקלטה אם קיימת הקלטה
- ליד כל הערה: אם יש הקלטה — כפתור **נגן** ליד כפתור המחיקה

## עריכה
- לוחצים על גוף ההערה (הטקסט והשורה עם התאריך) כדי לערוך
- בעריכה אפשר לשנות טקסט, להקליט הקלטה חדשה (מחליפה את הקיימת אם יש), או להסיר הקלטה קיימת בכפתור קטן עם סימון ליד שורת ההקלטה
- שמירה מעדכנת את ההערה; ביטול מבטל את השינויים במצב העריכה

## מחיקה
- כפתור המחיקה מוחק את **כל** ההערה (טקסט והקלטה יחד)
- לפני מחיקה מוצגת שאלה לאישור

## הסרת הקלטה בלבד
- בעריכה, אם יש הקלטה — אפשר להסיר רק אותה
- אם להערה אין טקסט בכלל, הסרת ההקלטה תמחק את כל ההערה (אחרי אישור)

## הודעות
- אחרי שמירה מוצגת הודעה שהפעולה הצליחה
- אם משהו נכשל מוצגת הודעת שגיאה
